# 火眼智擎—汽配领域知产保护分析助手

精准靶向汽配领域侵犯知识产权犯罪的智能分析平台。

## 技术栈

- **后端**: FastAPI + Peewee + SQLite
- **前端**: 独立部署（基于后端API）

## 环境要求

- Python 3.8+

## 快速启动

```bash
# 安装依赖
pip install fastapi uvicorn peewee pandas openpyxl python-docx

# 启动服务
python main.py

# 访问 API 文档
http://localhost:8000/docs
```

## 项目结构

```
.
├── api/              # API 路由层
├── models/           # 数据模型
├── services/         # 业务逻辑层
├── utils/            # 工具函数
├── data/             # 数据存储
├── config/           # 配置文件
└── main.py           # 应用入口
```

## 主要功能

- **数据导入**: Excel/CSV 文件导入资金流水、通讯记录、物流记录
- **案件管理**: 案件的增删改查
- **智能分析**: 主观明知评分、敏感词检测、角色推断
- **关联分析**: 上下游产业链分析、跨案关联、累犯检测
- **数据台账**: 人员和交易台账管理
- **报告生成**: Word/TXT 分析报告导出

## API 接口

| 模块 | 路径 | 说明 |
|------|------|------|
| 数据导入 | `/api/upload/*` | 上传资金流水/通讯/物流 |
| 案件管理 | `/api/cases/*` | 案件 CRUD |
| 可疑线索 | `/api/cases/{id}/suspicious` | 可疑线索查询 |
| 智能分析 | `/api/analyze/*` | 证据分析、全案分析 |
| 关联分析 | `/api/relations/*` | 上下游关系、跨案关联 |
| 数据台账 | `/api/ledger/*` | 人员和交易台账 |
| 数据导出 | `/api/export/csv` | CSV 导出 |
| 报告生成 | `/api/report/*` | 生成分析报告 |

详细接口文档请访问 `http://localhost:8000/docs`

## 核心算法

### 主观明知评分

评分公式：
- 直接承认假冒类 × 5（上限10分）
- 回避性话术类 × 4（上限8分）
- 暗示非正品类 × 2（上限4分）
- 其他类别各 ×1
- 价格低于正品50% +3分
- 无授权证明 +3分
- 无正规进货凭证 +2分

评分等级：
- 8-10分：高度可疑
- 5-7分：中度可疑
- 0-4分：低度可疑

### 敏感词库

7类233+敏感关键词，覆盖：
- 直接承认假冒类
- 回避性话术类
- 暗示非正品类
- 销售行为类
- 生产/贴牌类
- 授权状态类
- 品牌词库

### 刑事门槛

| 罪名 | 违法所得 | 非法经营 |
|------|----------|----------|
| 假冒注册商标罪 | ≥3万 | ≥5万 |
| 假冒两种以上 | ≥2万 | ≥3万 |
| 情节特别严重 | ≥15万 | ≥25万 |
